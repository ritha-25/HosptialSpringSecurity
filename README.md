# Hospital Management System

A RESTful backend API built with Spring Boot that handles the core operations of a hospital — registration, authentication, patient management, and doctor management.

---

## What this project does

A hospital registers itself into the system. When it does, an admin account is automatically created for it. That admin can then create doctors and manage patients. Patients register themselves and manage their own profiles. 
Everyone logs in through the same endpoint and gets back a JWT token that controls what they can and cannot do.
